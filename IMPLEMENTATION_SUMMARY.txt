📋 USAGE WORKFLOW
═══════════════════════════════════════════════════════════════════════════════

TEACHER WORKFLOW:
1. Open app → Click "Teacher Dashboard"
2. Fill: Exam Name, Title, Time, Max Grade
3. Set number of subjects (1-10)
4. For each subject:
   - Enter subject name
   - Upload Excel/CSV with questions
   - Optional: set question count (use N of imported M)
   - Check/uncheck shuffle toggles
5. Optional: Upload student whitelist
6. Click "Create Quiz"
7. Copy 6-digit ID or magic link → Share with students
8. Monitor results:
   - Click "View Results" on quiz
   - See submissions, search, filter
   - Export to Excel
   - View topic analysis
   - Download individual PDFs

STUDENT WORKFLOW:
1. Open app → Click "Student - Join Quiz"
2. Enter 6-digit code OR paste magic link
3. Enter name & email
4. Read instructions, check "I accept the rules"
5. Click "Begin Assessment"
   - Fullscreen activates
   - Timer starts
   - Webcam may activate (if enabled)
   - Don't switch tabs!
6. Answer questions (vertical scroll, any order)
7. Change answers if needed
8. Click "Finish & Submit"
9. See instant results:
   - Score & percentage
   - Topic breakdown
   - Peer ranking (if enabled)
   - Print PDF or request review

✅ Tab switching (5-sec warning → auto-submit)

✅ Text selection
✅ Duplicate submissions per email
✅ Whitelist bypass (if configured)

WHAT IS NOT BLOCKED (Why):
⚠️ Physical screenshots (OS-level) → Solution: watermark + webcam snapshots
⚠️ Phone camera photos → Same deterrent
⚠️ Screen recording → Monitored by webcam

BEST PRACTICES:
✓ Use in proctored environments (classroom/testing center)
✓ Enable webcam for additional deterrent
✓ Whitelist students for high-stakes exams
✓ Use in low-stakes formative assessments with supervision
